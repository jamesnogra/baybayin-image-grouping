# Local Installation
 - Install required packages `pip install flask`
 - Put unsorted/ungrouped images inside `static/temp_images`
 - Run the command `python app.py` and then open in browser `http://127.0.0.1:8885/`
 - In the browser, select a character from the dropdown, then check images checkbox, and then click the "Move Images" button
 - The grouped images are stored at `static/grouped`
